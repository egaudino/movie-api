# Movie-Api
> Api to create and query Movies

## Installation

Build:
```sh
mvn clean install
```
## Usage example

```sh
mvn spring-boot:run
```

## Access in browser

```sh
http://localhost:8080/swagger-ui.html
```

## Stack
- Java 8
- Spring Boot 2
- Embedded MongoDB
- Lombok
- Swagger2
- Junit & Mockito
